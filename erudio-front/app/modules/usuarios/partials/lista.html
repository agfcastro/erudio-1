Usuário

Definição:
Usuário é uma pessoa habilitada para acessar o sistema. Cada usuário possui ser próprio nome de usuário e senha.

Instruções Rápidas:
Clique no botão + para cadastrar um Usuário.
Clique no nome de um Usuário para editá-lo.
Clique no ícone da lixeira para remover um Usuário.

Definição:
ex: Escolas, Universidades, Secretarias de Educação, etc...

Dica:
Clique sobre o item desejado para ver as opções disponíveis para ele.
