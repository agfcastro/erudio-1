Permissão

Definição:
Permissões são os níveis de acesso que cada usuário pode ter, de acordo com a função que exerce, podendo ver e/ou alterar informações dos módulos do sistema.

ex: Permissão para alunos os habilitam a ver as notas, mas professores têm a permissão de também incluí-las.

Instruções Rápidas:
Pesquise por um usuário cadastrado que deseja cadastrar/alterar permissões.
Clique sobre o nome de um Usuário para gerenciar suas permissões.
Clique na lixeira para remover todas as Permissões de um Usuário.

Definição:
ex: Escolas, Universidades, Secretarias de Educação, etc...

Dica:
Clique sobre o item desejado para ver as opções disponíveis para ele.
