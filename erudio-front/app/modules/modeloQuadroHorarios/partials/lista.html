Modelo de Horário

Definição:
Modelos de horários definem a quantidade e a duração das aulas bem como a duração do intervalo para um determinado curso.

Instruções Rápidas:
Clique no botão + para cadastrar um Modelo de Horário.
Clique no nome de um Modelo de Horário para editá-lo.
Clique na lixeira para remover um Modelo de Horário.

Definição:
ex: Escolas, Universidades, Secretarias de Educação, etc...

Dica:
Clique sobre o item desejado para ver as opções disponíveis para ele.
