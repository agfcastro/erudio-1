Cargo

Definição:
São funções exercidas por servidores públicos.

Ex: Diretor, Secretária escolar, etc

Instruções Rápidas:
Clique no botão + para cadastrar um Cargo.
Clique no nome de um Cargo para editá-lo.
Clique no ícone da lixeira para remover um Cargo.
