Tipo de Unidade

Definição:
Tipos de Unidades são formas de classificação de uma unidade de ensino conforme as características do alunos atendidos, idade, localização, etc.

ex: Centro de Educação Infantil, Grupo Escolar, Escola Básica, Escola Isolada, etc...

Instruções Rápidas:
Clique no botão + para cadastrar um Tipo de Unidade.
Clique sobre o nome de um Tipo de Unidade para editá-la.
Clique na lixeira para remover Tipo de Unidade.
